<h1 align="center">DBunny</h1>

<p align="center">Cliente desktop para bancos de dados PostgreSQL.</p>

---

## Tecnologias

**Frontend**
- React (Vite)
- TypeScript
- Tailwind CSS
- Monaco Editor
- TanStack Table

**Backend**
- Tauri (Rust)

## Como rodar

Requisitos: **Node.js**, **npm** e **Rust**

```bash
# instalar dependências
npm install

# rodar em modo dev
npm run dev
```
